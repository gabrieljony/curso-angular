Manipulating Arrays

O que são Arrays?

No JavaScript, Array faz parte dos objetos globais sa linguagem e é utilizado para armazenar uma coleção de elementos em uma única variável. E os arrays nos fornecem vários métodos para facilitar a sua manipulação. Os arrays também são conhecido como listas.

  • .findIndex

    O método array.findIndex() retorna o índice do primeiro elemento do array que satisfaz a função de teste fornecida. Caso contrário ou caso não encontre o elemento, retorna -1, indicando que nenhum elemento passou no teste.

  • .fill

    O método array.fill() altera todos os elementos em uma matriz para um valor estático, de um índice inicial (padrão 0) para um índice final (padrão array.length). Ele retorna a matriz modificada.

    Novo: {{ arrayExampleFind }}

  • .some

    O método array.some() testa se pelo menos um(1) elemento do array passa no teste implementado pela função fornecida. Ele retorna verdadeiro(true) se, o array, encontrar um elemento para o qual a função fornecida retorna verdadeiro(true); Caso contrário, retorna falso(false). Esse método NÃO modifica o array.

    Verdadeiro ou Falso: {{ arrayExampleFind }}

  • .every

    O método array.every() testa se todos os elementos da matriz passam no teste implementado pela função fornecida. Ele retorna um valor booleano. CAso passe por todos testes retorna true, caso contrario retorna false.

  • .push

    Utilizamos o array.push() para adicionar um elemento ao FINAL do array.

  • .includes

    Utilizamos o método array.includes(), onde nos retorna se um array contém um determinado elemento ou não. O seu retorno é boolean true ou false.
