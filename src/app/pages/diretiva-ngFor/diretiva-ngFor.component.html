Diretiva *ngFor

O *ngFor é uma diretiva de repetidor - uma forma de apresentar uma lista de itens.

Define um bloco de HTML como um único item deve ser exibido e, em seguida, diz ao Angular para usar esse bloco como um modelo para renderizar cada item na lista.

No exemplo abaixo foi definido o bloco da tag li, logo será repetido esse bloco

https://v10.angular.io/api/common/NgForOf

NgFor é implementado pela diretiva NgForOf.

Local variables

  • index: number: The index of the current item in the iterable.
  • count: number: o comprimento do iterável.
  • first: boolean: Verdadeiro quando o item é o primeiro item no iterável.
  • last: boolean: Verdadeiro quando o item é o último item no iterável.
  • even: boolean: Verdadeiro quando o item tem um índice uniforme no iterável.
  • odd: boolean:Verdadeiro quando o item tem um índice ímpar no iterável.
